# RUNBOOK

> **Papel:** índice dos rituais operacionais com julgamento humano no loop. Se é
> só rodar um comando (sem decisão), não é runbook — é linha de README ou `make`.
> Se um agente executa sozinho, é recipe (`docs/conventions/recipes/`). Cada
> ritual vive em `docs/runbooks/<slug>.md` (copie `_TEMPLATE-RUNBOOK.md`).

## Rituais

| Ritual | Quando | Runbook |
|--------|--------|---------|
| <nome> | <trigger> | [docs/runbooks/<slug>.md](docs/runbooks/<slug>.md) |
